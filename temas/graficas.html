2. Anatom√≠a de Matplotlib

El n√∫cleo de Matplotlib gira en torno a la pareja Figure (fig) y Axes (ax). Una Figure es el lienzo global; los Axes son los sistemas de coordenadas donde dibujas.

Diagrama sugerido de la jerarqu√≠a Figure ‚Üí Axes ‚Üí Axis/Spines.

C√≥digo 2.1 ‚Äî Crear un lienzo vac√≠o 
fig = plt.figure(figsize=(6,4))      # Crea la Figure
ax  = fig.add_subplot(111)          # A√±ade un Axes (1x1 grid)
ax.set_title("Ejes primarios")
ax.plot([],[])                      # A√∫n sin datos
plt.show()

üìë Glosario r√°pido

fig

Lienzo global (puede contener varios Axes).

ax

Sistema de coordenadas donde se dibujan l√≠neas, barras, etc.

subplot

Atajo para crear Axes multipanel en grillas.

Spines

Los bordes del gr√°fico (superior, inferior, izq., der.).

3. Tipos de Gr√°ficos üç≠

Matplotlib soporta una gran variedad de gr√°ficos. Estos son los m√°s usados:

L√≠nea (plot)

Datos continuos, series temporales.

Barra (bar)

Categor√≠as discretas.

Dispersi√≥n (scatter)

Pares (x,y) para correlaciones.

Histograma (hist)

Distribuci√≥n de frecuencias.

C√≥digo 3.1 ‚Äî Cuatro gr√°ficos en un solo lienzo

# 2) Barras

axs[0,1].bar(cats, [3,7,5], color='teal')
axs[0,1].set_title('Barras')

# 3) Dispersi√≥n

axs[1,0].scatter(rng.random(50), rng.random(50), c='tomato')

# 4) Histograma

plt.tight_layout() # Ajusta el espaciado para evitar superposiciones

5. El Poder de fig, ax = plt.subplots()

Esta funci√≥n devuelve la dupla Figure y Axes de forma directa, favoreciendo un c√≥digo m√°s limpio y expl√≠cito. Es la forma recomendada de iniciar tus gr√°ficos.

Expl√≠cito

Separa claramente fig y uno o m√°s ax.

Multipanel

Crea grillas al instante (ej: plt.subplots(2,3)).

tight_layout()

Optimiza espacios autom√°ticamente en la figura.

C√≥digo 5.1 ‚Äî Ejemplo con subplots() 
# Crear una figura con 1 fila y 2 columnas de subplots
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 4))

# Primer subplot
x = np.linspace(0, 10, 100)
ax1.plot(x, np.sin(x), color='crimson')
ax1.set_title('Seno en ax1')

# Segundo subplot
ax2.plot(x, np.cos(x), color='dodgerblue')
ax2.set_title('Coseno en ax2')

plt.tight_layout() # Importante para que los subplots no se solapen
plt.show()

7.1 ‚Äî Movimiento Parab√≥lico

g = 9.81      # Aceleraci√≥n gravitacional (m/s¬≤)

10. Personalizaci√≥n Detallada: Dando Vida a tus Gr√°ficos üé®‚ú®

Crear un gr√°fico b√°sico es solo el primer paso. El verdadero poder de Matplotlib radica en su incre√≠ble capacidad para personalizar cada detalle de tus visualizaciones. Una personalizaci√≥n cuidadosa no solo embellece tus gr√°ficos, sino que los hace mucho m√°s claros y efectivos para comunicar tus hallazgos cient√≠ficos. ¬°Vamos a explorar c√≥mo convertir un gr√°fico simple en una obra de arte informativa!

10.1 ax.plot(): Dibujando L√≠neas con Estilo y Precisi√≥n

La funci√≥n ax.plot(x, y, ...) es tu principal herramienta para gr√°ficos de l√≠neas. Adem√°s de los datos x e y, puedes controlar cada detalle visual:

  • color (o c): Define el color de la l√≠nea (ej: 'blue', 'red', 'green', c√≥digos hexadecimales como '#FF5733', o tuplas RGB/RGBA como (0.1, 0.2, 0.5)).
  • linewidth (o lw): Controla el grosor de la l√≠nea (un n√∫mero flotante, ej: 2.5).
  • linestyle (o ls): Define el estilo de la l√≠nea. Opciones comunes:
    • '-' o 'solid': L√≠nea s√≥lida (por defecto).
    • '--' o 'dashed': L√≠nea discontinua.
    • '-.' o 'dashdot': L√≠nea de guion y punto.
    • ':' o 'dotted': L√≠nea punteada.
  • marker: Especifica el s√≠mbolo a usar para cada punto de datos (ej: 'o', 's', '^', 'x', '+').
  • markersize (o ms): Tama√±o de los marcadores.
  • markerfacecolor (o mfc), markeredgecolor (o mec): Colores del marcador.
  • label: Etiqueta de la l√≠nea que se usar√° en la leyenda del gr√°fico.
C√≥digo 10.1.1 ‚Äî Personalizando una trayectoria de proyectil (F√≠sica) 
# (Reutilizando par√°metros de la secci√≥n 7.1)

        label=f'Trayectoria (v0={v0}m/s, Œ∏={angulo_grados}¬∞)')

ax.set_title('Lanzamiento de Proyectil Detallado üéØ', fontsize=16, color='#7f1d1d')
ax.set_xlabel('Distancia Horizontal (m)', fontsize=12, color='#7f1d1d')
ax.set_ylabel('Altura Vertical (m)', fontsize=12, color='#7f1d1d')
ax.grid(True, linestyle=':', alpha=0.6)
ax.legend(fontsize=10)
plt.show()

10.8 ax.grid(): A√±adiendo una Rejilla de Referencia

Mejora la legibilidad facilitando la estimaci√≥n de valores.

  • ax.grid(True): Activa la rejilla principal.
  • which: 'major' (defecto), 'minor', o 'both'.
  • axis: 'x', 'y', o 'both' (defecto).
  • alpha: Transparencia (0.0 a 1.0).
C√≥digo 10.8.1 ‚Äî Gr√°fico de una funci√≥n f√≠sica con rejilla detallada 
distancia_r = np.linspace(1, 10, 100)
k_const = 8.9875e9; carga_Q = 1e-6 
campo_E = (k_const * carga_Q) / (distancia_r**2)

fig, ax = plt.subplots(figsize=(8,5))
ax.plot(distancia_r, campo_E, color='firebrick', lw=2)
ax.set_title('Intensidad del Campo El√©ctrico vs. Distancia $E = kQ/r^2$ ‚ö°', fontsize=14, color='#7f1d1d')

ax.grid(which='major', linestyle='-', linewidth='0.6', color='darkgrey', alpha=0.7)
ax.minorticks_on() # Activar ticks menores para la rejilla menor
ax.grid(which='minor', linestyle=':', linewidth='0.4', color='lightgrey', alpha=0.6)

ax.set_yscale('log') # Escala logar√≠tmica en Y
plt.show()

10.9 ax.scatter(): Visualizando Puntos de Datos Individuales

Ideal para mostrar la relaci√≥n entre dos variables o datos experimentales.

  • s: Tama√±o de los marcadores.
  • c: Color (puede ser un array para colores variables, usando `cmap`).
  • alpha: Transparencia.
  • cmap: Mapa de colores si c es num√©rico (ej. 'viridis', 'plasma').
C√≥digo 10.9.1 ‚Äî Dispersi√≥n de datos experimentales (F√≠sica/Estad√≠stica) 
rng = np.random.default_rng(0)
num_datos = 70
voltaje_V = rng.uniform(1, 10, num_datos)
corriente_A = 0.2 * voltaje_V + rng.normal(0, 0.3, num_datos) # Ley de Ohm con ruido
resistencia_calculada = voltaje_V / corriente_A
tamano_puntos = rng.uniform(15, 150, num_datos)

fig, ax = plt.subplots(figsize=(9,6))
dispersion = ax.scatter(voltaje_V, corriente_A, s=tamano_puntos, c=resistencia_calculada, 
                        cmap='plasma', alpha=0.75, edgecolor='black', linewidth=0.6)

ax.set_title('Relaci√≥n Voltaje-Corriente (Datos Simulados) üí°', fontsize=14, color='#7f1d1d')
cbar = fig.colorbar(dispersion, label='Resistencia Calculada (Œ©)')
ax.grid(True, linestyle=':', alpha=0.5)
plt.show()

10.10 ax.axis(): Atajos para el Control de Ejes

Forma conveniente de controlar varios aspectos de los ejes a la vez.

  • ax.axis('equal'): Escalas iguales en X e Y.
  • ax.axis('tight'): Ajusta l√≠mites a los datos.
  • ax.axis('off'): Oculta ejes.
  • ax.axis([xmin, xmax, ymin, ymax]): Establece l√≠mites.
C√≥digo 10.10.1 ‚Äî Demostrando ax.axis() 
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 4.5))
