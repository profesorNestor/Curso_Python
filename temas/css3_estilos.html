    Controla el orden de apilamiento (profundidad) de los elementos posicionados. Un elemento con un z-index mayor se muestra encima de uno con un z-index menor. Solo afecta a elementos con position distinto de static. auto es el valor por defecto.
    float
    left | right | none
    Hist√≥ricamente usado para layouts (ej: im√°genes flotando junto a texto). Hace que un elemento "flote" a la izquierda o derecha, permitiendo que otros contenidos fluyan a su alrededor. Para layouts complejos, Flexbox y Grid son preferibles hoy en d√≠a. Requiere manejo de clearfix.
    clear
    left | right | both | none
    Se usa para controlar el comportamiento de los elementos que siguen a elementos flotantes. clear: both; es com√∫n para asegurar que un elemento comience debajo de cualquier elemento flotante anterior.
