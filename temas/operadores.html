Los operadores aritm√©ticos se utilizan para realizar operaciones matem√°ticas b√°sicas con n√∫meros. [cite: 90] Funcionan de manera muy similar a como lo hacen en matem√°ticas. [cite: 91]

Los operadores relacionales se utilizan para comparar dos valores. [cite: 96] El resultado de una comparaci√≥n siempre es un valor booleano: True (verdadero) o False (falso). [cite: 97] Son esenciales para las estructuras de control como los condicionales (if, elif, else) y los bucles (while). [cite: 97]

5. Operadores de Asignaci√≥n

Ya conoces el operador de asignaci√≥n b√°sico =. Python ofrece operadores de asignaci√≥n combinados que realizan una operaci√≥n aritm√©tica y luego asignan el resultado a la variable original. Son una forma abreviada y eficiente de actualizar variables.

Operador Ejemplo Equivalente a Descripci√≥n
= x = 10 x = 10 Asigna el valor de la derecha a la variable de la izquierda.
+= x += 5 x = x + 5 Suma el valor de la derecha a la variable y asigna el resultado a la variable.
-= x -= 3 x = x - 3 Resta el valor de la derecha de la variable y asigna el resultado.
*= x *= 2 x = x * 2 Multiplica la variable por el valor de la derecha y asigna el resultado.
/= x /= 4 x = x / 4 Divide la variable por el valor de la derecha y asigna el resultado (flotante).
//= x //= 3 x = x // 3 Realiza divisi√≥n entera y asigna el resultado.
%= x %= 2 x = x % 2 Calcula el m√≥dulo y asigna el resultado.
**= x **= 3 x = x ** 3 Eleva la variable a la potencia del valor de la derecha y asigna el resultado.

6. Precedencia de Operadores (Orden de las Operaciones)

Al igual que en matem√°ticas, los operadores en Python tienen un orden de precedencia que determina cu√°les se eval√∫an primero en una expresi√≥n compleja. Por ejemplo, la multiplicaci√≥n y divisi√≥n se realizan antes que la suma y la resta.

El orden general (de mayor a menor precedencia) es aproximadamente:

  1. () Par√©ntesis (para agrupar y forzar un orden).
  2. ** Exponenciaci√≥n.
  3. *, /, //, % Multiplicaci√≥n, Divisi√≥n, Divisi√≥n entera, M√≥dulo (se eval√∫an de izquierda a derecha).
  4. +, - Suma y Resta (se eval√∫an de izquierda a derecha).
  5. <, <=, >, >=, !=, == Operadores de comparaci√≥n.
  6. not Operador l√≥gico NOT.
  7. and Operador l√≥gico AND.
  8. or Operador l√≥gico OR.

resultado1 = 5 + 3 * 2  # Primero 3*2=6, luego 5+6=11
print(f"5 + 3 * 2 = {resultado1}")

resultado2 = (5 + 3) * 2 # Primero (5+3)=8, luego 8*2=16
print(f"(5 + 3) * 2 = {resultado2}")
