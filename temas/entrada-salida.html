1. Comunicaci√≥n con el Programa: Entrada y Salida (E/S)

Imagina que tu programa Python es un cient√≠fico brillante üßë‚Äçüî¨. Para que pueda hacer su trabajo, necesita dos cosas fundamentales:

  • üì¨ **Recibir informaci√≥n (Entrada):** Datos de experimentos, par√°metros de un modelo, respuestas de un usuario, etc.
  • üì¢ **Presentar resultados (Salida):** Conclusiones, gr√°ficos, mensajes informativos, alertas, etc.

En Python, las herramientas principales para esta comunicaci√≥n son la funci√≥n input() para la entrada de datos desde el teclado, y la funci√≥n print() para la salida de datos a la pantalla o consola. En entornos interactivos como Google Colab o Jupyter Notebooks, tambi√©n contamos con funciones especiales como display() para mostrar resultados de forma m√°s enriquecida. ¬°Vamos a dominarlas! üöÄ

2. Mostrando Informaci√≥n: La Vers√°til Funci√≥n print()

La funci√≥n print() es tu principal aliada para mostrar mensajes, valores de variables y resultados de tus c√°lculos en la consola. Es fundamental para entender qu√© est√° haciendo tu programa y para comunicarte con el usuario.

Uso B√°sico:

Simplemente pasas lo que quieres imprimir como argumento entre los par√©ntesis. Puedes imprimir texto (cadenas o strings), n√∫meros, o el valor de variables. 

print("¬°Hola, futuros cient√≠ficos de datos!") # Imprime una cadena de texto
print(3.14159)                             # Imprime un n√∫mero flotante
numero_atomico_H = 1
print(numero_atomico_H)                  # Imprime el valor de una variable (1)

# Puedes imprimir m√∫ltiples cosas a la vez, separadas por comas:
nombre_elemento = "Ox√≠geno"
simbolo = "O"
print("El elemento", nombre_elemento, "tiene el s√≠mbolo:", simbolo)
# Salida: El elemento Ox√≠geno tiene el s√≠mbolo: O

Por defecto, print() a√±ade un espacio entre los argumentos separados por comas y un salto de l√≠nea (\n) al final de cada llamada.

Personalizando la Salida con Par√°metros Clave:

‚ú® El Par√°metro sep (Separador):

El par√°metro sep te permite especificar qu√© cadena de texto se usar√° para separar los m√∫ltiples argumentos que pasas a print(). Por defecto, es un espacio. 

dia = 25
mes = "Diciembre"
anio = 2024
print(dia, mes, anio)                     # Salida: 25 Diciembre 2024 (separador por defecto: espacio)
print(dia, mes, anio, sep="/")            # Salida: 25/Diciembre/2024
print("Masa", "Volumen", "Densidad", sep=" | ") # Salida: Masa | Volumen | Densidad
print("Puntos:", 10, 20, 30, sep=", ")      # Salida: Puntos: 10, 20, 30

üèÅ El Par√°metro end (Finalizador):

El par√°metro end controla qu√© car√°cter o cadena se imprime al final de la llamada a print(). Por defecto, es un salto de l√≠nea (\n), lo que hace que cada print() aparezca en una nueva l√≠nea. Puedes cambiarlo si quieres que varias llamadas a print() contin√∫en en la misma l√≠nea. 

print("Procesando datos", end="")      # No a√±ade salto de l√≠nea
print(".", end="")
print(".", end="")
print(". ¬°Completo!")
# Salida: Procesando datos... ¬°Completo!

print("Resultado 1:", end=" | ")
print("Resultado 2:", end=" | ")
print("Resultado 3")
# Salida: Resultado 1: | Resultado 2: | Resultado 3

üí° Dato para Colab: En Google Colab, la salida de cada celda se muestra debajo de ella. Usar end="" es √∫til si quieres construir una l√≠nea de salida a partir de varios print() dentro de la misma ejecuci√≥n de celda.

3. F-Strings: Formateo de Texto Elegante y Moderno

Las **f-strings** (o cadenas literales formateadas) son una forma moderna, concisa y muy legible de incrustar expresiones y valores de variables directamente dentro de una cadena de texto en Python (disponibles desde Python 3.6). ¬°Son la forma preferida de formatear cadenas para la mayor√≠a de los casos!

Para crear una f-string, simplemente precede la comilla de apertura de tu cadena con la letra f o F. Luego, dentro de la cadena, puedes colocar variables o expresiones entre llaves { }, y Python reemplazar√° esas llaves con el valor evaluado de la expresi√≥n. 

nombre_planeta = "Marte"
distancia_media_sol_km = 227.9e6  # 227.9 millones de km
numero_lunas = 2

# Usando f-string para crear un mensaje
mensaje_cientifico = f"El planeta {nombre_planeta} tiene {numero_lunas} lunas y est√° a una distancia media de {distancia_media_sol_km} km del Sol."
print(mensaje_cientifico)
# Salida: El planeta Marte tiene 2 lunas y est√° a una distancia media de 227900000.0 km del Sol.

# Incluso puedes hacer c√°lculos dentro de las llaves:
masa_kg = 70
altura_m = 1.75
imc = masa_kg / (altura_m ** 2)
print(f"Paciente con masa {masa_kg} kg y altura {altura_m} m. Su IMC es: {imc}")
# Salida: Paciente con masa 70 kg y altura 1.75 m. Su IMC es: 22.857142857142858

Formateando N√∫meros con F-Strings:

Las f-strings tambi√©n te permiten controlar el formato de los n√∫meros, como la cantidad de decimales, el uso de notaci√≥n cient√≠fica, o el relleno con ceros. Esto se hace a√±adiendo dos puntos : seguidos de un especificador de formato dentro de las llaves. 

PI = 3.141592653589793
radio = 5.0
area = PI * radio**2

# Mostrar con 2 decimales:
print(f"El valor de PI es aproximadamente: {PI:.2f}")        # Salida: El valor de PI es aproximadamente: 3.14
print(f"El √°rea de un c√≠rculo con radio {radio} es: {area:.2f} unidades cuadradas.") # Salida: ...78.54...

# Notaci√≥n cient√≠fica con 3 decimales:
constante_planck = 6.62607015e-34
print(f"Constante de Planck (h) ‚âà {constante_planck:.3e} J¬∑s") # Salida: ...6.626e-34 J¬∑s

# Relleno con ceros (ej. para c√≥digos o IDs):
id_experimento = 7
print(f"ID del Experimento: {id_experimento:04d}") # Salida: ID del Experimento: 0007

üé® Ventaja de las F-Strings: Son m√°s legibles y a menudo m√°s r√°pidas que otros m√©todos de formateo de cadenas en Python (como el m√©todo .format() o el operador %). ¬°Intenta usarlas siempre que puedas!

4. Recibiendo Datos del Usuario: La Funci√≥n input()

Para que tus programas sean interactivos, necesitas una forma de que el usuario ingrese datos. En Python, la funci√≥n input() se encarga de esto. Cuando se llama, input() pausa la ejecuci√≥n del programa y espera a que el usuario escriba algo en la consola y presione Enter.

Uso B√°sico:

Puedes pasarle un mensaje (prompt) como argumento a input(), que se mostrar√° al usuario para indicarle qu√© informaci√≥n debe ingresar. 

# Pedir el nombre al usuario
nombre_usuario = input("Por favor, ingresa tu nombre: ")
print(f"¬°Hola, {nombre_usuario}! Encantado de conocerte. üëã")

# Pedir una magnitud f√≠sica
fuerza_aplicada_texto = input("Introduce el valor de la fuerza aplicada (en Newtons): ")
print(f"Has ingresado una fuerza de: {fuerza_aplicada_texto} N")

¬°Importante! input() Siempre Devuelve una Cadena (str)

Un detalle crucial es que la funci√≥n input() siempre devuelve el dato ingresado por el usuario como una cadena de texto (string), sin importar si el usuario escribe n√∫meros. 

edad_texto = input("¬øCu√°ntos a√±os tienes? ") # Si el usuario escribe 20
print(type(edad_texto))                      # Salida:  ¬°Es "20", no el n√∫mero 20!

# Si intentas hacer una operaci√≥n matem√°tica directamente, ¬°tendr√°s un error!
# edad_mas_cinco = edad_texto + 5 # Esto dar√≠a un TypeError

# Debes convertir la cadena a un tipo num√©rico si quieres hacer c√°lculos:
edad_numero = int(edad_texto)   # Convertir a entero
print(f"En cinco a√±os tendr√°s: {edad_numero + 5} a√±os.")

# Para n√∫meros con decimales, usa float():
altura_texto = input("Ingresa tu altura en metros (ej: 1.75): ")
altura_numero = float(altura_texto)
print(f"Tu altura multiplicada por dos es: {altura_numero * 2} m")

üîÑ Conversi√≥n de Tipos: Recuerda usar int() para convertir a entero y float() para convertir a n√∫mero decimal (flotante) las entradas que esperas sean num√©ricas. Si el usuario ingresa algo que no se puede convertir (ej. texto cuando esperas un n√∫mero), Python generar√° un error ValueError.

5. Visualizaci√≥n Enriquecida con display() (Colab/Jupyter)

En entornos interactivos como Google Colab y Jupyter Notebooks, adem√°s de print(), tienes acceso a la funci√≥n display(). Esta funci√≥n es m√°s potente porque puede renderizar diferentes tipos de objetos de forma m√°s "inteligente" y visualmente atractiva. Proviene del m√≥dulo IPython.display.

¬øC√≥mo usar display()?

Primero, necesitas importarla. Luego, puedes pasarle varios tipos de objetos: 

from IPython.display import display, HTML, Markdown, Latex

# Display b√°sico (similar a print, pero a veces con mejor formato por defecto)
nombre_cientifico = "Albert Einstein"
display(nombre_cientifico)

# Display de DataFrames de Pandas (se ven como tablas bonitas)
# import pandas as pd
# data = {'Columna1': [1, 2], 'Columna2': [3, 4]}
# df = pd.DataFrame(data)
# display(df) # En Colab, esto renderiza una tabla HTML interactiva

# Display de HTML (¬°puedes generar salidas con estilo!)
html_output = "<h3 style='color: blue;'>Resultados del Experimento üî¨</h3>"
html_output += "<p>La muestra A mostr√≥ un incremento del <strong>25%</strong>.</p>"
display(HTML(html_output))

# Display de Markdown
markdown_text = """
### Secci√≥n Importante
- Punto 1
- Punto 2 con `c√≥digo`
"""
display(Markdown(markdown_text))

# Display de LaTeX (renderiza f√≥rmulas matem√°ticas)
formula_latex = r"$$ E = mc^2 $$" # La r indica "raw string"
display(Latex(formula_latex))

(Recuerda que para los ejemplos de Pandas, HTML, Markdown y LaTeX, necesitas importar las clases correspondientes como se muestra).

üåü Ventaja de display() en Colab/Jupyter: Es especialmente √∫til para:

  • Mostrar tablas de datos (DataFrames de Pandas) de forma interactiva.
  • Incrustar HTML personalizado para salidas con mucho estilo.
  • Renderizar texto formateado con Markdown directamente en la salida.
  • Mostrar im√°genes, audio, video y, por supuesto, f√≥rmulas LaTeX.
Para la salida de texto simple, print() suele ser suficiente y m√°s universal si tu c√≥digo se ejecutar√° fuera de un notebook.

Matem√°ticas: √Årea de un Rect√°ngulo

Solicitar la base y altura de un rect√°ngulo, calcular y mostrar su √°rea.
